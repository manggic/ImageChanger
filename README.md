
# Image Changer is a simple react app

## It contains three main element :
<br>

1. **Previous button** :- One can move to the previous image on click. If the current image is the first image then on clicking one can see a toast message pop-up saying  `Current image is the First image`. 
2. **Image Body** :- The image itself  which changes on button click.
3. **Next button** :- One can move to the next image on Click. If the current image is Last image then on clicking a toast message will pop-up saying  `Current image is the Last image `
